# Douceurs & Gourmandises 🍰

Projet de site web pour une pâtisserie artisanale haut de gamme.

## Structure du Projet
- `index.html` : Accueil et présentation.
- `patisseries.html` : Liste des créations artisanales.
- `commande.html` : Formulaire de réservation en ligne.
- `contact.html` : Informations de contact et boutique.
- `css/` : Styles personnalisés (Vieux rose, Chocolat, Crème).
- `js/` : Scripts pour l'interactivité.
- `images/` : Dossiers pour organiser les visuels (hero, produits, boutique, icons).

## Installation
Ouvrez simplement `index.html` dans votre navigateur pour visualiser le prototype.

## Personnalisation
Les images sont actuellement des placeholders stylisés via CSS. Vous pouvez ajouter vos propres fichiers `.jpg` dans les dossiers correspondants du répertoire `images/`.
